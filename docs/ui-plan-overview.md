## Известные UX-проблемы (исправлено)

- **Бледный текст меток** (UI-4) — `--ox-fg-dim` поднят до `#b0bec5`, `--ox-fg-muted` до `#78909c`. Метки в `PlanetStat` переведены на `fg-dim` вместо `fg-muted`.
- **Чёрный экран после пересборки Docker** (UI-4) — версионирование localStorage (`STORAGE_VERSION=2`) + `RootErrorBoundary`.
- **Белые поля при автозаполнении** (UI-4) — `-webkit-box-shadow inset` trick в `app.css`.
