# Шаблон промпта для серии ремастера 64-77

**Применение:** скопируй структуру ниже, замени `{{...}}` на
плана-специфические значения. Включи блоки из `_blocks/` через
явное копирование (Markdown не поддерживает include).

**Когда использовать:**
- Новый план серии — копируешь шаблон полностью.
- Continuation-промпт — берёшь только нужные секции (КОНТЕКСТ +
  ЧТО ДЕЛАТЬ + ОБЯЗАТЕЛЬНЫЕ ПРАВИЛА + GIT-ИЗОЛЯЦИЯ).

---
