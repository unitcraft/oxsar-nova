# Экспедиция

Экспедиция (миссия 15) — разведка неизвестной системы. Флот летит в
заданные координаты, проводит там время (hours), с вероятностями
выпадает один из 12 исходов.

## Требования

- Минимум **50 000 metal-eq** во флоте (примерно 10 Small Transporter'ов).
- Водород на полёт — как для обычного транспорта.
- Astrophysics (в будущем — ограничит количество одновременных экспедиций).

## Исходы

Веса зависят от `exp_power = expo_tech + hours×2 + spy_tech/10 × probes^0.4`.

| Исход | Смысл |
|---|---|
| resource | Ресурсы (M+Si+H) по трюму. Зажимаются 3×стоимости флота. |
| asteroid | Ресурсы (M+Si) по трюму. |
| ship | Флот присоединил LF или Recycler. |
| battlefield | Бой с остатками чужого флота, лут. |
| credit | Кредиты (если hours ≥ 4). |
| artefact | Случайный артефакт в коллекцию (hours ≥ 4). |
| xSkirmish | Бонус к исследованию (hours ≥ 3). |
| fast | Ускорение возврата на 10–60% времени полёта. |
| delay | Задержка возврата на 10–30% времени полёта. |
| nothing | Ничего. |
| **lost** | **Флот уничтожен полностью.** Вероятность растёт с exp_power. |

## Anti-farm

- **min_fleet**: флот должен стоить ≥ 50k metal-eq.
- **reward_cap**: resource/asteroid награды зажимаются `3 × стоимости флота`.
- **lost_weight**: растёт с `exp_power` — чем сильнее флот, тем выше
  шанс потерять всё.
