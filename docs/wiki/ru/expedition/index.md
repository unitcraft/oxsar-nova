Экспедиция — разведка неизвестных координат. Флот летит
туда, проводит там указанное время (часы), затем возвращается
с одним из 12 возможных исходов.

## Требования

- Флот должен стоить не менее **50 000 ед. металла** в эквиваленте.
  Слишком слабый флот не сможет отправиться в экспедицию.
- Водород на полёт — как при обычном транспорте.
- Чем выше [[unit:13]] (астрофизика), тем больше одновременных
  экспедиций можно вести.

## Возможные исходы

Вероятность каждого исхода зависит от `сила_экспедиции`:
`тех_экспедиция + часы×2 + тех_шпионаж/10 × число_зондов^0.4`

| Исход | Что происходит |
|---|---|
| Ресурсы | Металл, кремний, водород по трюму флота |
| Астероид | Металл и кремний по трюму |
| Флот | К экспедиции присоединяется [[unit:31]] или [[unit:37]] |
| Сражение | Бой с обломками чужого флота, лут ресурсов |
| Кредиты | Валюта (доступно при часы ≥ 4) |
| Артефакт | Случайный артефакт (часы ≥ 4) |
| Исследование | Бонус к одной из технологий (часы ≥ 3) |
| Ускорение | Возврат на 10–60% быстрее обычного |
| Задержка | Возврат на 10–30% медленнее обычного |
| Ничего | Флот возвращается без результата |
| **Гибель** | **Весь флот уничтожен.** Вероятность растёт с сила_экспедиции |

## Баланс наград

- Ресурсные награды ограничены: не более **3 × стоимости флота**.
  Это исключает фарм экспедиций огромным флотом.
- Вероятность гибели также растёт с силой флота — нельзя послать
  [[unit:40]] (Звезду смерти) и ждать лёгкой победы.

## Тактика

Оптимальный флот — несколько [[unit:31]] для трюма и [[unit:38]]
для повышения бонуса тех_шпионаж. Часы = 3–4 открывают максимум исходов
при разумном расходе топлива.
