# Начало игры

oxsar-nova — это браузерная MMO-стратегия в духе OGame. Вы начинаете с одной
планеты, строите экономику, исследуете технологии, снаряжаете флоты и
участвуете в PvP с другими игроками.

## Первые 30 минут

1. **Постройте шахту металла до уровня 5.** Металл — основа всего.
2. **Постройте кремниевый лабораторный комплекс до уровня 3.** Нужен для
   исследовательского центра.
3. **Постройте солнечную электростанцию.** Без энергии шахты замедляются.
4. **Постройте исследовательскую лабораторию до уровня 1.** Открывает
   технологии.
5. **Начните исследование энергии.** Ускоряет всё.
