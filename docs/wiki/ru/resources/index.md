# Ресурсы

В oxsar-nova три базовых ресурса плюс энергия.

- **Металл** — добывается на шахтах. Основа для любой постройки.
- **Кремний** — добывается в кремниевых лабораториях. Нужен для техники
  и кораблей.
- **Водород** — добывается в водородных заводах. Используется для полётов
  флота и продвинутых построек.
- **Энергия** — не хранится, только производится. Если потребление выше
  производства, шахты замедляются пропорционально дефициту.

## Хранилища

У каждой планеты есть ограничение на хранение ресурсов — объём склада.
При заполнении склада добыча конкретного ресурса останавливается до
освобождения места (перевод, экспорт, расход на постройку).

## Курс рынка

Ориентировочный обмен: 1 металл = 0.5 кремния = 0.33 водорода. Точные
курсы см. в [Экономике](../economy/).
