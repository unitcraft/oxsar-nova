## 1. Контекст

Legacy [`Officer.class.php::hireOfficer`](../../projects/game-legacy-php/src/game/page/Officer.class.php):
```php
$of_time = ($off_res["of_{$off_id}"] == 0)
    ? time() + (7*60*60*24)            // новый: now + 7 дней
    : $off_res["of_{$off_id}"] + (7*60*60*24);  // продление: текущий expire + 7 дней
```

Если у игрока активная подписка не истекла, `hireOfficer` **продлевает**
её на 7 дней от текущего expire (срок суммируется). Origin
[`Activate`](../../projects/game-nova/backend/internal/officer/service.go)
возвращает `ErrAlreadyActive` и блокирует операцию. Комментарий в коде
утверждает обратное («legacy тоже не даёт продлить за credit»), но это
не соответствует коду legacy.

Также найдены два места `folder=13` в officer (auto_renew notify +
expire notify) — folder=13 не существует в legacy const, после
миграции 0087 не входит в message_folders справочник → не пройдёт
FK constraint. Включаю в этот подплан фикс на folder=12 (SYSTEM).
