Цель — довести UI oxsar-nova до полного функционального паритета с легаси oxsar2,
но не копировать слепо: там где легаси неудобен, делаем лучше.

Легаси: http://localhost:8080 (PHP/Yii 1.1, curl-анализ апрель 2026)
Наш UI: http://localhost:5173 (React + Vite)
