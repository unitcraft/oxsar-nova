**Дата**: 2026-05-01

## 1. Контекст

Legacy [`Chat.class.php`](../../projects/game-legacy-php/src/game/page/Chat.class.php)
имеет:
1. **BBCode**: `[b]`, `[i]`, `[u]`, `[s]`, `[color=...]`, `[:emoji:]`,
   `[img]`, `[url]`, `[url=...]` парсятся в HTML.
2. **Read-only checkRO**: блок отправки если `umode=true` или
   `observer=true`.
3. **history**: SELECT 75 сообщений `ORDER BY messageid DESC`.
4. **Anti-flood same-as-last**: `if($message != $mes["message"])`
   (если последнее сообщение этого юзера такое же — не сохраняется).
5. **Tag-balance Teg()**: парные `[b]/[/b]` валидируются (одиночные
   убираются).
6. **Moderation::mask**: уже есть в backend (план 46).

Origin `ChatScreen`:
- (1) BBCode → plain text (simplification P72.S4.BBCODE).
- (2) umode/observer-RO **отсутствует**.
- (3) historyLimit = 50 (не 75).
- (4) anti-flood **отсутствует**.
- (5) tag-balance — n/a (нет BBCode).
- (6) Moderation::mask — есть.

## 2. Что нужно

### Backend

#### `chat/handler.go::Send` (POST или WS)
- Прочитать `users.umode, is_observer` для author. Если umode || observer
  → вернуть `400 Bad Request` с error.code='chat_read_only'.
- Anti-flood: SELECT последнее сообщение автора в этом канале; если
  body совпадает — отбросить (не возвращать ошибку, легаси молча
  игнорировал).
- historyLimit: 50 → 75.

### Frontend

#### `ChatScreen.tsx`
- Если /me (`useMeQuery`) показывает `umode || observer` → input
  disabled + текст «(read-only: vacation/observer mode)».
- BBCode-парсер: компактный (regex-based, не TipTap), поддержка
  `[b][i][u][s]` + `[url]http://...[/url]` + `[url=...]text[/url]` +
  `[color=...]...[/color]` + `[:emoji_name:]` (image из
  `/assets/origin/emo/<name>.gif`). Output — React-fragments с
  `<b>`/`<i>`/`<u>`/`<s>`/`<a>`/`<span style>`/`<img>`. SAFE: только
  whitelist-теги, без `dangerouslySetInnerHTML`.
- Toolbar (опционально, не блокирующее): кнопки B/I/U/S вокруг
  выделения. На первой итерации — без toolbar, BBCode пишется руками.

### i18n
- `chat.readOnlyVacation` — «Чат недоступен в режиме отпуска» / EN.
- `chat.readOnlyObserver` — «Чат недоступен наблюдателю».

### Тесты
- `chat/handler_test.go` (новый): umode-RO 400, anti-flood
  (повторное body — no-op).
- BBCode parser unit-test.
