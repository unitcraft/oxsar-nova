# История обновлений Oxsar

## 0.5.0 — 2026-04-28

- Spring 4 origin-фронта: 13 экранов (друзья, сообщения, чат, заметки,
  поиск, настройки, офицеры, профессии, соглашение, история обновлений,
  поддержка, калькулятор технологий).
- Полное OpenAPI-расширение для всех 14 экранов Spring 4.
- Уточнение DTO Officer и Profession под фактический backend.

## 0.4.0 — 2026-04-28

- Spring 3 origin-фронта: 7 экранов (артефакты, инфо-страницы,
  древо технологий, рекорды, рейтинг).
- Catalog endpoints для зданий / юнитов / артефактов / древа.

## 0.3.0 — 2026-04-28

- Spring 2 origin-фронта: 17 экранов (альянс, рынок, ремонт,
  статистика боёв, операции флота).
- Полная имплементация alliance-домена в origin-фронте.

## 0.2.0 — 2026-04-28

- Spring 1 origin-фронта: 7 главных экранов (главная, постройки,
  исследования, верфь, галактика, миссии, империя).

## 0.1.0 — 2026-04-28

- Bootstrap origin-фронта: Vite + React + TanStack Query + Zustand.
- 3-frame layout (header / left-menu / content / footer).
